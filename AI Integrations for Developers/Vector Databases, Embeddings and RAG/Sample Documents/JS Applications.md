# JS Applications

In the "JS Applications" course, students will learn what HTTP requests are and how to use them. They will become familiar with REST services, what a BaaS (Backend as a Service) is and how to work with it, what asynchronous code means (Promises, using async/await), and what Templating and Routing are. During the course, they will create a Single Page Application using the techniques learned from previous lectures. They will understand the architecture of an application and how to properly structure their applications. Toward the end of the course, they will explore various design patterns and their practical applications, create their own web components using the Web Components standard, and set up a Webpack environment from scratch. Additionally, 30% of the exercise tasks will be solved with the help of AI in order to encourage the use of modern technologies for process automation, while simultaneously developing skills for the effective application of AI tools in real-world conditions.
